# Demonstração Metamorphose Fit

Arquivos:

- `metamorphose-fit-demo.webm`: vídeo demonstrativo final.
- `metamorphose-fit-demo.vtt`: legendas/capítulos do vídeo.
- `player.html`: página para reproduzir o vídeo com capítulos clicáveis.

Para assistir, abra `player.html` no navegador ou reproduza diretamente o arquivo `.webm`.
